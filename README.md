# Todo Manager - Java 21

Projeto desenvolvido para estudo de arquitetura backend utilizando 
Java 21, Maven, JUnit e integração com banco de dados.

O objetivo é implementar um sistema simples de gerenciamento de tarefas (Todo),
aplicando boas práticas de desenvolvimento, testes automatizados e organização de código.

--

## Tecnologias utilizadas:

- Java 21 (LTS)
- Maven (gerenciamento de dependências)
- JUnit 5 (testes automatizados)
- JDBC (acesso ao banco de dados)

--

## Como executar o projeto

1. Clonar o repositório

git clone https://github.com/seu-usuario/seu-repo

2. Entrar na pasta

cd todo-manager

3. Compilar o projeto com Maven

mvn clean install

--

## Testes

O projeto utiliza **JUnit 5** para testes automatizados.

Para executar os testes:

mvn test
